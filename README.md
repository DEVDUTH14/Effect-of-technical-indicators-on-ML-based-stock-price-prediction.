# Effect-of-technical-indicators-on-ML-based-stock-price-prediction.
Compared 2 sets of technical indicators (Group1: RSI,SMA,MACD,Stochastic; Group2: ADX,ATR,OBV,Bollinger) on LSTM models for banking stocks (HDFC,SBI,JPM,BAC) using Yahoo Finance data. Group1 yielded lower MSE/MAE (e.g. JPM:0.0006 vs 0.0021) across most banks, outperforming Group2 in price prediction accuracy.
